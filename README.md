# InvestIT Robo Advisor

InvestIT is a goal-based robo-advisory platform developed using Python and Dash. It helps investors evaluate their financial health, determine risk tolerance, assess investment goals, and receive personalized portfolio recommendations.

The platform integrates risk profiling, mutual fund ranking, multi-asset portfolio construction, Monte Carlo simulation, SIP planning, and historical backtesting into a single interactive dashboard. Users can analyze their probability of achieving financial goals, explore recommended investments, and visualize portfolio growth through intuitive charts and analytics.

## Key Features

* Risk Profiling Engine
* Financial Health Assessment
* Goal-Based Investment Planning
* SIP Requirement Calculator
* Monte Carlo Goal Simulation
* Mutual Fund Ranking System
* Multi-Asset Portfolio Construction
* Historical Portfolio Backtesting
* Interactive Dash Dashboard
* Portfolio Allocation & Performance Analytics
